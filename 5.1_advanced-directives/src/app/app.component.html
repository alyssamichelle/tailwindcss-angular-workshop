Guess what?! You have already been using two of Tailwind's Directives this whole time! Directives in Tailwind are CSS at rules (starting with an `@`). These tell CSS how to behave. `@charset`,`@import`,`@namespace` are three built into CSS. Here is the list of available Tailwind CSS Directives:

  • @tailwind
  • @apply
  • @layer
@tailwind
@apply
@layer
